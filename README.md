# Gym App

A Vite + React application for tracking workouts, progress, and gym insights.

## Getting started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```

## Deployment

This project is ready to be pushed to GitHub and deployed with any static hosting service that supports Vite builds.
